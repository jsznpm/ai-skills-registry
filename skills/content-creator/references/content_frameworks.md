# Content Frameworks Reference

## Blog Post Templates

### How-To / Tutorial
1. Hook — the pain point the reader has right now
2. Why it matters — cost of not solving it
3. Prerequisites / what you need
4. Step-by-step instructions (numbered, one action per step)
5. Common mistakes section
6. Recap + CTA

### Listicle
1. Hook framing the problem the list solves
2. Brief context on selection criteria
3. Numbered items, each with: sub-heading, 2-4 sentences, one example/visual
4. Comparison table (optional) if items differ across weighted dimensions
5. Closing recommendation + CTA

### Comparison / "X vs Y"
1. Hook — the decision the reader is stuck on
2. At-a-glance summary table
3. Deep dive per option (features, pricing, ideal use case)
4. Head-to-head verdict by use case ("best for teams", "best for solo")
5. CTA pointing to the recommended path

### Thought Leadership / Opinion
1. Bold claim or contrarian take in the first two sentences
2. Evidence — data, case study, or firsthand experience
3. Counterargument acknowledged and addressed
4. Implication — what the reader should do differently
5. CTA inviting discussion or offering a related resource

## Content Pillar Ratio (40/25/25/10)

Use this split when planning a content calendar:

- **40% Educational** — how-tos, explainers, best practices
- **25% Promotional** — product features, case studies, testimonials
- **25% Engagement** — questions, polls, behind-the-scenes, UGC
- **10% Experimental** — new formats, trending topics, one-off tests

## Repurposing Matrix

One long-form piece → multiple assets:

| Source | Repurpose into |
|---|---|
| Blog post | Twitter/X thread (5-8 posts), LinkedIn carousel, email newsletter section |
| Blog post | Short-form video script (60-90 sec hitting the top 3 points) |
| Webinar/podcast | Quote graphics, audiogram clips, blog post transcript-summary |
| Case study | Sales one-pager, LinkedIn post, testimonial snippet |
| Data/research | Infographic, stat-of-the-day social posts |

## Headline Formulas

- "How to [achieve outcome] without [common pain point]"
- "[Number] ways to [achieve outcome]"
- "The [adjective] guide to [topic]"
- "Why [common belief] is wrong (and what to do instead)"
- "[Outcome] in [timeframe]: a step-by-step plan"
