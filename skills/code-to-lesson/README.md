# code-to-lesson

Analyzes a code project end-to-end and writes a deep-dive engineering tutorial
to `code_tutorial.md` — architecture, folder structure, design patterns
(Refactoring.guru), React patterns (Patterns.dev), Bullet Proof React practices,
dependency/version audit, and anti-pattern refactors.

Install:
```bash
skillpool add code-to-lesson
```
