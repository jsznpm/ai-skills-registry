# Examples

## 1. Poster request

> "Make me a poster for a jazz night at a small basement bar."

Don't jump to layout. First write a design philosophy `.md` — name a movement
(e.g. "Smoke Geometry"), 4-6 paragraphs on form, color, rhythm, craftsmanship.
Deduce the subtle reference (the intimacy and improvisation of a basement set).
Then express it as a single-page `.pdf`/`.png`: limited palette, thin design-
forward type, repeating forms, nothing overlapping, generous margins. Refine a
second pass for museum-grade crispness.

## 2. Studio identity

> "A visual for a minimalist ceramics studio."

Philosophy leans quiet — vast negative space, paper texture, whisper-quiet
labels. Output: one restrained composition where the few words are integrated as
visual elements, not typeset captions.

## 3. Multi-page request

> "Give me a 4-page set for an album zine."

Page one establishes the philosophy. Pages 2-4 are distinct twists on the same
movement — variations that tell a tasteful story. Bundle as one `.pdf` (or
multiple `.png`s).

## Output contract
- Always two artifacts minimum: the philosophy `.md` + the canvas `.pdf`/`.png`.
- Original work only — never reproduce existing artists' designs.
- Text minimal, visual-first; everything inside the canvas with breathing room.
